## 🚀 How to Use SEO Dashboard

### **Step 1: Open SEO Dashboard**
1. Go to `seo-dashboard.html` in your browser
2. Or click "🔍 SEO Dashboard (NEW)" from Settings page sidebar

### **Step 2: View SEO Status**
- **Dashboard** shows all pages with their SEO scores
- ✅ Green = Working perfectly
- ⚠️ Yellow = Has warnings
- ❌ Red = Has critical errors

### **Step 3: Edit SEO for a Page**
1. Select a page from dropdown (e.g., "Home Page")
2. Edit:
   - **Page Title** (30-60 characters optimal)
   - **Meta Description** (50-160 characters optimal)
   - **Meta Keywords** (5-10 main keywords)
   - **OG Image URL** (for social media sharing)
   - **Canonical URL** (official page URL)
   - **Robots Directive** (index/noindex)

### **Step 4: Check SEO Health**
- Click **"✅ Check SEO"** button
- See errors and warnings
- Fix issues before downloading

### **Step 5: Download Updated HTML**
- Click **"📥 Download HTML"** button
- File downloads to your computer
- **Upload this file to GitHub**

### **Step 6: Upload to GitHub**
1. Go to your GitHub repository
2. Upload the downloaded HTML file
3. Commit changes with message like:
   ```
   Update SEO for Home Page
   ```
4. Wait 1-2 minutes for GitHub Pages to update
5. Check live site!

## 🎯 SEO Best Practices

### **Page Title**
- Keep between 30-60 characters
- Include main keyword at beginning
- Add brand name "Hunny Collection PK"
- Make it unique for each page

### **Meta Description**
- Keep between 50-160 characters
- Include primary and secondary keywords
- Write compelling copy that encourages clicks
- Include call-to-action when possible

### **Meta Keywords**
- Focus on 5-10 main keywords
- Use comma-separated values
- Include location-based keywords (Pakistan, Lahore, etc.)
- Don't keyword stuff

### **OG Image**
- Use 1200x630px images for best results
- Include brand colors and logo
- Use JPG or PNG format
- Host on reliable CDN or GitHub

### **Robots Directive**
- **index, follow** - Allow search engines (for main pages)
- **noindex, follow** - Don't index but follow links (for account pages)
- **noindex, nofollow** - Block completely (for checkout, admin)
