## 🛍️ **Products Management**

**Access:** Admin Panel → Products

### **What You Can Do:**

#### **Add New Product:**
1. Click "+ Add Product"
2. Fill in details:
   - Product Name
   - Price (Rs.)
   - Category (select or create new)
   - Description
   - Sizes (S, M, L, XL)
   - Colors (Red, Blue, etc.)
   - **Images** (Upload or paste URLs)
3. Click "Save Product"

#### **Edit Product:**
1. Find product in table
2. Click "Edit" button
3. Change any details
4. Click "Save Product"

#### **Delete Product:**
1. Find product in table
2. Click "Delete" button
3. Confirm deletion

#### **Image Upload:**
**Option 1: Direct Upload**
- Click "Choose Files"
- Select images
- Click "📤 Upload Images"
- Wait for upload

**Option 2: External URLs**
- Upload to ImgBB/Cloudinary
- Copy direct links
- Paste in "Image URLs" field
- Comma separated for multiple

## 📦 **Orders Management**

**Access:** Admin Panel → Orders

### **What You Can Do:**

#### **View All Orders:**
- See all customer orders
- Filter by status
- View order details

#### **Update Order Status:**
1. Find order
2. Click on order to view details
3. Click status button:
   - ✅ **Confirm** - Order confirmed
   - 🚚 **Ship** - Order shipped
   - ✅ **Deliver** - Order delivered
   - ❌ **Cancel** - Order cancelled

#### **Contact Customer:**
1. Find order
2. Click WhatsApp button (💬)
3. Pre-filled message opens
4. Send message

## 🔍 **SEO Settings**

### **What is SEO?**
SEO = Search Engine Optimization
Helps your store appear in Google search

### **How to Optimize:**

#### **Meta Description:**
```
Shop premium female fashion at Hunny Collection PK. 
Dresses, Kurtis, Jewelry & more. Cash on Delivery. 
Fast shipping across Pakistan.
```

**Tips:**
- Keep it under 160 characters
- Include keywords
- Make it compelling

#### **Meta Keywords:**
```
female fashion, dresses, kurtis, jewelry, bags, 
shoes, pakistan, online shopping, cash on delivery
```

**Tips:**
- Use comma separation
- Include relevant keywords
- Don't spam

#### **OG Image:**
- Size: 1200x630 pixels
- Format: JPG or PNG
- Shows when sharing on Facebook/Twitter
