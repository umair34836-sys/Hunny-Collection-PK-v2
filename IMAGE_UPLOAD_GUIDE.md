## 📝 **Step-by-Step Guide**

### **Step 1: Create ImgBB Account**

1. Go to: **https://imgbb.com/**
2. Click **"Sign Up"** (top right)
3. Enter:
   - Email
   - Password
   - Username
4. Verify email
5. **Done!** ✅

---

### **Step 2: Upload Images**

1. **Login** to ImgBB
2. Click **"Upload"** button
3. **Select images** (multiple select kar sakte ho)
4. **Upload** click karo
5. Wait for upload to complete

---

### **Step 3: Get Direct Links**

1. Uploaded image par click karo
2. **"Direct link"** option dhundo
3. **Copy** the link
4. Example link:
   ```
   https://i.ibb.co/abc123/image.jpg
   ```

**Pro Tip:** Multiple images ke links copy karo!

---

### **Step 4: Add to Admin Panel**

1. **Admin panel** kholo
2. **Add Product** click karo
3. Scroll down to **"Product Images"** section
4. **"Or paste image URLs"** field mein paste karo
5. **Comma separated** multiple URLs:
   ```
   https://i.ibb.co/abc123/image1.jpg, https://i.ibb.co/def456/image2.jpg, https://i.ibb.co/ghi789/image3.jpg
   ```
6. **Save Product** click karo

**Done!** ✅

## 💡 **Pro Tips**

### **Image Optimization:**

**Before Upload:**
- Resize to 1000x1000px (max)
- Compress with TinyPNG
- Format: JPG or PNG
- Size: Under 500KB

**Tools:**
- TinyPNG: https://tinypng.com/
- Squoosh: https://squoosh.app/

---

### **Multiple Images:**

**Product ke liye:**
1. Front view
2. Back view
3. Side view
4. Detail/close-up
5. Model wearing (if available)

**Order:**
- Pehli image = Main/Thumbnail
- Baaki images = Gallery

---

### **Naming Convention:**

**Good:**
```
dress-red-front.jpg
dress-red-back.jpg
dress-red-detail.jpg
```

**Bad:**
```
IMG_001.jpg
IMG_002.jpg
```
